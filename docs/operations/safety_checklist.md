# Safety Checklist

Run this checklist before hardware movement tests.

- [ ] Motors are elevated or wheels are not touching the ground.
- [ ] E-STOP is accessible.
- [ ] Correct power supply is connected.
- [ ] Raspberry power is separate from servos and motors.
- [ ] GND is common between signaling boards.
- [ ] Servo limits are configured.
- [ ] Motor speed is limited.
- [ ] Motor command duration is limited.
- [ ] Cables are reviewed and labeled.
- [ ] No loose wires can touch motor terminals.
- [ ] Firmware prints ready events.
- [ ] Heartbeat behavior is understood.
- [ ] Another person nearby knows how to cut power during risky tests.
